README

    The pokemon battle simulator V.2.1 is a simulator pokemon game for the pokemon lover. As a simulator the game will only use five pokemons. In other to 
    choose those five pokemon, we did a survey among some fan like you. After the survey we were able to choose the five best. To know more about those
    pokemon go to the pokemon website: http://dbz3220.s3-website-us-east-1.amazonaws.com/pokemon.html
 

    The website interface is pretty easy to navigate through. In the home page, you will be able to see the navigation area with four clickable button 
    that will send you to the title of that specifique button you press. As you probably find out those button are: Forum, View Battle, Pokedex and
    Credits. The following part will disect those button one by one.

FORUM

    As you know already the forum button will allow you to drop your comment and also talk about what you like about the simulator. Those feedback 
    forum will allow the dbz3220 TEAM to improve the simulator in other give you the best experience ever.

VIEW BATTLE

    This button is pretty much straight forward. In this part the user will be able to see simulator battle video of an older user experience.
    This video will show to the user how other player used the simulator.

POKEDEX
    This button is the big one because it contains the different pokemon with all their abilities. It is the little bible of the simulator.
    We recommend all the user to check it out.

CREDITS

    This button will give you the information about the dbz3220 TEAM and will also provide some pictures as well. 

START

    Now the user know all the necessary information, he/she can click on the START button that will allow him/her to start the simulator.
    
HELP/Secret

    In other to help the user to go back to the home page, we the dbz3220 add a clickable button called home page.
    The user can also click on the home page pikatchu picture to hear him out.
